如何写作业

  1. 向老师发送自己的 github 账号，加入公共项目
  2. clone 本项目到本地，在 homework文件夹下建自己的中文名字的文件夹
  3. 代码作业放在自己的文件夹里，如:
- 若愚
  - task1.html
  - task2-1.html 
  - task2-3.html
  - task4
    - imgs
    - js
    - css
    - task4-1.html
    - task4-2.html
  - task5.html
  - ...
